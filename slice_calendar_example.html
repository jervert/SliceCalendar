Slice Calendar

Inline calendar

Calendar is permanently shown (with no limits):

<div class="calendar-box">

 [...]
</div>

Calendar is permanently shown (with limits):

<div class="calendar-box" 
data-min="2012-04-10"
data-max="2012-06-15">
  [...]
</div>

Calendar in date type input

Calendar in date type input with no limits (Date 1):

<input type="date" />

Calendar in date type input with narrowing of selectable dates (date 2):

<input type="date" min="2012-11-20" max="2013-03-21" />

Calendario in text type input

Calendar in date type input with no limits (Date 3):

<input type="text" />

Calendar in date type input with narrowing of selectable dates (Date 4):

<input type="text" class="date-picker" 
data-min="2012-11-20" data-max="2014-03-21" />
